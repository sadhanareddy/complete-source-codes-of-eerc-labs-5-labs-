System Requirements:

The current version of the Virtual Lab applets pages requires a web browser and the Java plug-in (version 1.5.0_11 or later is recommended). The Java plug-in is part of the Java Runtime Environment (JRE)    . You may check whether Java is installed correctly by visiting the Test Your Java Version    page on the Sun web site. In addition to testing the Java installation, you will obtain information on the installed version of Java.

: Minimum Requirement
     * JRE    version 1.5 or later
     * OpenGL
     * Install Java 3D API

Order of software installation:

Strictly follow the installation of software in below order:

  1. Download the latest Java JRE    (prefered 1.6.0_11 or later ) if not available on your system
  2. Download the latest from Java 3D API
    or from
    Java 3D Downloads: Release Builds
  3. Close all browser
  4. Install JRE    .
  5. Install JAVA 3D on your system. For help read Instructional manual
  6. Open the browser again and view the recommended Java3D testing links.
  7. In case you are not able to see the applet remove all the previous installation and please try the fresh installation as above said

Trouble shooting:

  1. Java plug-in might not be installed properly
  2. Order of installation might be wrong.
  3. More than one version of Java on your machine
  4. JRE/JDK version problem
  5. Cache problem. Close all the browser and remove the cache and start again.
  6. Insufficient memory
